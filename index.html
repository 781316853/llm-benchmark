阅读说明

  • 「综合分」= 旧三基准(DeepSWE/Vibe Code/llm2014)归一化均值×80% + 新两基准(SWE-bench Pro+Terminal-Bench)归一化均值×20%,满分100;新两榜无数据时不惩罚(权重回流至旧三榜)。列表默认按综合分降序排列。
  • 「命中」表示该模型在对应基准组中进入榜单,4/4 表示五项基准(四个基准组)均有排名(SWE-Pro 与 Terminal-Bench 合并计为1组)。
  • DeepSWE 与 Vibe Code 为百分比;llm2014 为 10 分制综合分(由等级数值折算);SWE-bench Pro 为 Scale SEAL 标准化榜单 Pass@1(731题,顶级~59%,远难于 Verified);Terminal-Bench 2.1 为终端任务准确率(89题,取该模型最优 agent 成绩)。
  • 「—」表示该榜未收录此模型。DeepSWE 列分数后的 v1.1/v1.0 标识表示数据来源版本(v1.1=当前每日刷新,v1.0=历史榜单补充)。
  • 浅绿底色行表示跨榜命中数 ≥ 2 的模型;模型名后 NEW 表示近 7 天内首次上榜(仅基于 DeepSWE/Vibe/llm2014 三基准判定)。
  • 点击表头可按该列排序(按某评测排序时仅显示该评测有数据的模型)。
  • 「显示全部模型」开关可展开完整模型列表(包含仅出现在单一榜单的模型);默认仅显示命中≥2个基准组的模型。
  • 本页仅用于学习与对比展示,数据为快照,不代表对各模型能力的绝对评价。
